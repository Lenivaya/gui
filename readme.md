## Hotkeys

| Hotkey              |      Action      |
| ------------------- | :--------------: |
| ALT+D               |  Go to diagram   |
| ALT+J               |  Go to diagram JSON representation   |
| ALT+PLUS            |     Add node     |
| ALT+R               |   Run diagram    |
